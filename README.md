# Coding_Platform_API

## Instructions For Local Setup
1. Clone the Repo <br>
<pre> 
  git clone https://github.com/Jatin6256/Coding_Platform_API.git
</pre>
2. Install the dependencies
<pre> 
  npm i
</pre>
3. start the server
<pre>
  node index.js
</pre>
